About

The Maraton engine is a collection of modules that together form a sort of base for all of our projects. The engine is built with performance and freedom in mind allowing the developer to have full control of all the different parts of the engine.

The engine is written using a subset of C++. Featuring a 2D renderer, a intermediate-mode graphical user interface system and custom platform layers.
